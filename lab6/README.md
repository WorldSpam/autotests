# Sixth lab

Tested api of [GoRest](https://gorest.co.in/)

## How to use:
- Pull
- run `npm install` in lab6 directory
- Enter your token in test.js
- Run `npm test` to do the magic
- If any leftovers of past tests are present, find and delete them
